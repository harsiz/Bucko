# Changelog

- Initial release
- BDL scripting engine with full expression support
- Dialogue system with trigger matching, priority resolution, chaining
- Mood system (4D: energy, patience, chaos, warmth)

- Affection system with anti-exploit curve (0–1,000,000 internal / 0–1000 displayed)

- Console system with 30+ commands

- Autocomplete input with trigger label search
- Discord Rich Presence (optional, silent fail)
- In-memory cache with file watcher and cache warmup
- First-launch disclaimer and setup flow
- 20+ base game dialogue blocks
- Expression system with fallback ASCII faces
